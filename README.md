# Fitinera Financial Calculator - Notebook Examples

This repository stores various example usages of the [Fitinera financial calculator](https://github.com/samrom3/financial-itinera-calculator) Python library.

## Overview

The examples in this repository are primarily presented as Jupyter Notebooks. This format is ideal for showcasing the capabilities of the `fitinera` library, as it allows for:

-   **Programmatic Specification:** Easily define and customize financial simulations.
-   **Seamless Visualization:** Integrate charts and graphs to visualize simulation results.
-   **Interactive Exploration:** Manipulate parameters and interact with the data for a hands-on learning experience.

## Getting Started

To run these examples, you will need to have Python and Jupyter Notebook installed. You will also need to install the `fitinera` library:

```bash
pip install fitinera
```

Then, you can clone this repository and run the notebooks:

```bash
git clone https://github.com/your-username/fitinera-nb-examples.git
cd fitinera-nb-examples
jupyter notebook
```
